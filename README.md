## Usage
\\\
py motifs_interactive.py --image 1.jpg --model large
\\\

## Controls
| Key | Action |
|-----|--------|
| p | Preview mesh in Open3D |
| e | Edit parameters |
| d | Show depth map and mask |
| s | Save STL |
| r | Re-run depth inference |
| n | Skip image |
| q | Quit |
